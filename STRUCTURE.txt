Project Structure
=================

metrics-collection/
  - collect_metrics.py      : Prometheus metrics collector
  - run_experiment.py       : Main experiment runner
  - README.txt              : Usage instructions

output/
  csv/                      : All CSV output files
  logs/                     : All log files (locust, collector)

load-testing/
  src/                      : Locust load test files
  scripts/                  : Helper scripts

microservices-demo/
  deploy/docker-compose/    : Sock Shop deployment files

Quick Start
===========

1. Run an experiment:
   python metrics-collection/run_experiment.py --pattern ramp --duration 10 --output ramp_10min.csv

2. Find outputs:
   - CSV: output/csv/ramp_10min.csv
   - Logs: output/logs/*.log

3. Expected rows: Duration (min) × 2 × 7 services
   - 1 min  = 14 rows
   - 10 min = 140 rows
   - 30 min = 420 rows
